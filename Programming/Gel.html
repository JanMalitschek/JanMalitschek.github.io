Gel

Gel (or Gel-Engine as it was titled originally) is a game engine library for C++ using OpenGL, GLFW, GLEW, OpenAL and GLM.
The Github repository for Gel can be found here

This is the current Gel Logo. It was different at first but I changed it at some point when I upgraded Gel from a header only library to a dynamic link library.

This is the old Gel-Engine Logo

To put Gel to good use I created two major test projects recreating two classic games.

The project that I created for the old Gel-Engine - a Wolfenstein 3D clone.
It's not fully functional or anything but it has the basic features such as loading a level from a file, animated doors, shooting and enemies, which don't do much but can be killed.

The second project was to test the new Gel and was a Minecraft clone. Like before it's nowhere near complete and only features multithreaded chunk generation really.
