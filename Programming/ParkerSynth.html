Parker Synth (WIP)

Parker Synth is a Digital Modular Synthesizer VST Plugin.
It is supposed to enable the user to design a limitless variety of different sounds without being restricted by a fixed synthesizer setup.

The UI went through several design stages but this is the final one.

This is what the UI looked like originally.
